# portfolio-simple-website
I created a simple portfolio website using html and css.
This is a basic personal portfolio website created using HTML and CSS. The project is designed to showcase a clean layout and simple navigation while focusing on core front-end fundamentals.

## Pages Included
Home Page – Introduces the portfolio with a simple and responsive design.
Contact Page – Provides a way for visitors to get in touch using basic contact details.

## Technologies Used
HTML5
CSS3

## Purpose
This project was built to practice and demonstrate basic web development skills, including page structuring, styling, and navigation without using any frameworks or JavaScript.
